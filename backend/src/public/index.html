Authentication API

v1.0.0

Available Endpoints

POST /api/login
Authenticate user and create a session. Requires email and password in request body. Returns JWT token in HTTP-only cookie.
POST /api/register
Register a new user. Requires firstName, lastName, email, and password in request body. Creates user account and returns JWT token in HTTP-only cookie.
POST /api/logout
Logout current user by clearing authentication cookie.
GET /api/protected Auth Required
Get current authenticated user's information. Requires valid JWT token in cookie.
GET /health
Health check endpoint. Returns server status.
